# SDE-Rank (IJCS) — Reproducibility Package

This repository provides code and scripts to reproduce the experiments reported in our IJCS submission on **SDE-Rank** (Structural–Dynamic–Emotional Rank) for node influence ranking on communication/social networks.

## Repository structure
## 快速开始（3分钟复现）

### 1) 安装环境（Windows + Anaconda）
```bash
conda create -n sderank python=3.11 -y
conda activate sderank
pip install -r requirementss.txt


- `run_experiments.py` — main entry to run experiments, baselines, ablations, and sensitivity scans
- `sde_rank.py` — SDE-Rank implementation
- `baselines.py` — baseline influence ranking methods (PageRank, HITS, Katz, LeaderRank, etc.)
- `ic_simulation.py` — Independent Cascade (IC) simulation for influence estimation
- `generators.py` — synthetic generators for activity/sentiment signals
- `data/` — datasets (edge lists) and/or compressed copies
- `scripts/download_datasets.py` — optional dataset downloader (if included)
- `results/` or `结果/` — generated outputs (CSV/PNG)

> Note: Some public topologies do not provide message content or temporal activity logs. In such cases, activity and sentiment signals are generated by the provided synthetic generators (see manuscript for assumptions and limitations).
